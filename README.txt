Compile using gcc command:

	gcc -o moesi moesi.c

Execute using the command:
	./moesi

You can issue commands to the program the following way:

	XOY

	Where:
		X is a cache number from 0 to 2
		Y is a line number from 0 to 3
		O is an operation that can be 'r' for read or 'w' for write

	To finish the program you can type 'q'
